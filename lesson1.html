What is HTML?

HTML stands for HyperText Markup Language. It's the standard language used to create web pages. Think of HTML as the skeleton of a website - it provides the structure and content.

Key Concept: HTML uses "tags" to define different parts of your content. Tags are written inside angle brackets like <tag>

Basic HTML Structure

Every HTML document follows this basic structure:

<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <title>My First Web Page</title>
</head>
<body>
    <h1>Hello World!</h1>
    <p>This is my first paragraph.</p>
</body>
</html>

Let's break this down:

  • <!DOCTYPE html> - Tells the browser this is an HTML5 document
  • <html> - The root element that wraps all content
  • <head> - Contains meta information about the page (not visible on the page)
  • <title> - Sets the title shown in the browser tab
  • <body> - Contains all visible content on the page

Common HTML Tags

Headings

HTML has 6 levels of headings, from <h1> (most important) to <h6> (least important):

Example:

This is an H1 heading

This is an H2 heading

This is an H3 heading

This is an H4 heading

This is an H5 heading
This is an H6 heading

Paragraphs

Use <p> tags for paragraphs of text:

<p>This is a paragraph. It can contain multiple sentences 
and will automatically wrap to fit the width of the browser.</p>

Text Formatting

Example: 
<p>This text is <strong>bold</strong> and this is <em>italic</em>.</p>
<p>You can also use <mark>highlighted</mark> text.</p>

This text is bold and this is italic.

You can also use highlighted text.

Links

Links use the <a> tag with an href attribute:

<a href="https://www.google.com">Visit Google</a>
Visit Google

Images

Images use the <img> tag. Note: this tag is self-closing (no closing tag needed):

<img src="image.jpg" alt="Description of image" width="300">
Important: Always include the alt attribute for accessibility. It describes the image for screen readers and shows if the image fails to load.

Exercise 1: Create Your Profile Page

Your Task:

Below is a template. Modify it to create your own profile page with:

  1. Your name as an H1 heading
  2. A short paragraph about yourself
  3. An H2 heading "My Interests"
  4. A paragraph describing your hobbies
  5. A link to your favorite website

Instructions: Right-click this page and select "View Page Source" or press Ctrl+U (Cmd+Option+U on Mac) to see the HTML code. Copy the template below, modify it, and save as a new HTML file!

<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <title>My Profile</title>
</head>
<body>
    <h1>[Your Name]</h1>
    <p>[Write something about yourself here]</p>
    
    <h2>My Interests</h2>
    <p>[Describe your hobbies and interests]</p>
    
    <h2>Favorite Website</h2>
    <p>Check out my favorite website: 
       <a href="[URL]">[Website Name]</a>
    </p>
</body>
</html>
