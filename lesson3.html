What is CSS?

CSS stands for Cascading Style Sheets. While HTML provides the structure and content, CSS controls the visual appearance and layout. Think of HTML as the bones and CSS as the skin and clothing!

Key Concept: CSS works by selecting HTML elements and applying style rules to them. A CSS rule consists of a selector and a declaration block.

Three Ways to Add CSS

1. Inline CSS (Not Recommended)

Applied directly to HTML elements using the style attribute:

<p style="color: blue; font-size: 18px;">This is blue text.</p>

This is blue text.

2. Internal CSS (Good for Single Pages)

Written in the <head> section using <style> tags:

<head>
    <style>
        p {
            color: green;
            font-size: 16px;
        }
    </style>
</head>

3. External CSS (Best Practice!)

Written in a separate .css file and linked to your HTML:

<head>
    <link rel="stylesheet" href="styles.css">
</head>

In styles.css:

p {
    color: green;
    font-size: 16px;
}

CSS Selectors

Element Selector

Selects all elements of a specific type:

p {
    color: blue;
    font-size: 20px;
}

All paragraphs will be blue and 20px!

Class Selector

Selects elements with a specific class. Classes start with a dot (.) and can be reused:

<!-- HTML -->
<p class="highlight">This paragraph is highlighted.</p>
<p class="highlight">This one too!</p>

<!-- CSS -->
.highlight {
    background-color: yellow;
    padding: 5px;
}

This paragraph is highlighted.

This one too!

ID Selector

Selects a specific element with a unique ID. IDs start with a hash (#) and should be unique on the page:

<!-- HTML -->
<p id="special-para">This is special!</p>

<!-- CSS -->
#special-para {
    color: red;
    font-weight: bold;
}

This is special!

When to Use What:
  • Classes: For styling multiple elements the same way
  • IDs: For styling one unique element (also used for JavaScript)
  • Elements: For general styling of all elements of that type

Common CSS Properties

Colors

/* Color names */
color: red;
background-color: lightblue;

/* Hex codes */
color: #FF5733;
background-color: #3498db;

/* RGB values */
color: rgb(255, 87, 51);
background-color: rgba(52, 152, 219, 0.5); /* Last value is opacity */

Text Styling

Common text properties: 
p {
    color: navy;
    font-size: 18px;
    font-family: Arial, sans-serif;
    font-weight: bold;
    text-align: center;
    text-decoration: underline;
    line-height: 1.5;
}

Spacing

Two important properties for spacing:

  • Padding: Space inside the element (between content and border)
  • Margin: Space outside the element (between elements)
.demo-box {
    padding: 20px;        /* All sides */
    margin: 10px;         /* All sides */
    
    /* Or specify each side */
    padding-top: 10px;
    padding-right: 15px;
    padding-bottom: 10px;
    padding-left: 15px;
    
    /* Shorthand: top right bottom left (clockwise) */
    padding: 10px 15px 10px 15px;
}
This box has padding (inside) and margin (outside)!

Borders

div {
    border: 3px solid navy;
    border-radius: 10px;    /* Rounded corners */
    
    /* Or specify each property */
    border-width: 3px;
    border-style: solid;    /* solid, dashed, dotted */
    border-color: navy;
}

Width and Height

div {
    width: 300px;          /* Fixed width */
    height: 200px;         /* Fixed height */
    max-width: 100%;       /* Responsive: won't exceed container */
}

Exercise 1: Style a Simple Page

Your Task:

Create an HTML file with internal CSS that includes:

  1. An H1 heading styled with:
    • Color of your choice
    • Font size of 36px
    • Center alignment
  2. Paragraphs styled with:
    • Font family: Arial
    • Line height: 1.6
    • Color: dark gray (#333)
  3. A class called "important" that makes text:
    • Bold
    • Yellow background
    • Red text color

Exercise 2: Create a Styled Card

Your Task:

Create a "card" design using HTML and CSS:

  1. Create a div with class "card"
  2. Style it with:
    • Width: 300px
    • Padding: 20px
    • Border: 1px solid gray
    • Border radius: 10px
    • Background color: light blue
    • Box shadow: 0 4px 8px rgba(0,0,0,0.1)
  3. Inside the card, add a heading and paragraph with appropriate styling

Starter Template:

<!DOCTYPE html>
<html>
<head>
    <style>
        .card {
            /* Add your CSS here */
        }
    </style>
</head>
<body>
    <div class="card">
        <h2>Card Title</h2>
        <p>Card content goes here.</p>
    </div>
</body>
</html>
