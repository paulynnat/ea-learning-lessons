The CSS Box Model

Every HTML element is essentially a box. The CSS Box Model describes how the size of these boxes is calculated:

Margin (transparent, space outside the border)
Border (the edge of the box)
Padding (space inside the border)
Content

Box Model Properties

.box {
    width: 200px;          /* Content width */
    height: 100px;         /* Content height */
    padding: 20px;         /* Space inside the border */
    border: 5px solid navy;/* Border around padding */
    margin: 20px;          /* Space outside the border */
}
Content Area
(200px wide)

Total width calculation:
200px (content) + 40px (padding left+right) + 10px (border left+right) + 40px (margin left+right) = 290px total space

Pro Tip: Use box-sizing: border-box; to make width include padding and border: 
* {
    box-sizing: border-box;
}
/* Now width: 200px means the TOTAL width including padding and border! */

Shorthand Properties

/* All sides the same */
margin: 20px;
padding: 15px;

/* Vertical | Horizontal */
margin: 20px 40px;
padding: 10px 20px;

/* Top | Right | Bottom | Left (clockwise) */
margin: 10px 20px 10px 20px;

/* Individual sides */
margin-top: 10px;
margin-right: 20px;
margin-bottom: 10px;
margin-left: 20px;

Introduction to Flexbox

Flexbox is a powerful layout tool that makes it easy to align and distribute items in a container.

Basic Flexbox Setup

.container {
    display: flex;  /* This makes it a flex container */
}

.item {
    /* These are flex items (children of flex container) */
}

Flex Direction

Controls whether items flow in rows or columns:

Row (default): 
.container {
    display: flex;
    flex-direction: row;
}

Align Items (Vertical Alignment)

Controls how items are positioned along the cross axis:

Center (both horizontal and vertical): 
.container {
    display: flex;
    justify-content: center;
    align-items: center;
    height: 150px;
}
Perfectly Centered!

Flex Wrap

Controls whether items wrap to a new line:

.container {
    display: flex;
    flex-wrap: wrap;  /* Items will wrap to next line if needed */
}

Flex Property (for Items)

Controls how flex items grow or shrink:

.item-1 {
    flex: 1;  /* Takes 1 part of available space */
}
.item-2 {
    flex: 2;  /* Takes 2 parts (twice as much as item-1) */
}

Exercise 1: Create a Navigation Bar

Your Task:

Create a horizontal navigation bar using Flexbox:

  1. Create a container with class "navbar"
  2. Use display: flex
  3. Add 4 navigation items (Home, About, Services, Contact)
  4. Use justify-content: space-around
  5. Style with:
    • Background color: #333
    • Padding: 15px
    • Text color: white

Exercise 2: Create a Card Layout

Your Task:

Create a responsive card layout:

  1. Create a flex container with 3 cards
  2. Each card should have:
    • Width: 250px
    • Padding: 20px
    • Margin: 10px
    • Border: 1px solid gray
    • Border-radius: 8px
  3. Use flex-wrap: wrap so cards wrap on small screens
  4. Use justify-content: center

Exercise 3: Center a Box

Your Task:

Create a container that centers content both horizontally and vertically:

  • Container height: 300px
  • Use Flexbox to center a box inside
  • The centered box should have:
    • Width: 200px
    • Height: 100px
    • Background color of your choice
    • Some text inside

Hint: Use display: flex, justify-content: center, and align-items: center
